# NeurIPS 2026 E&D LaTeX Source

This source uses the official NeurIPS 2026 style file with the E&D option:

```latex
\usepackage[eandd]{neurips_2026}
```

The compiled submission PDF is `paper.pdf`. The authoritative OpenReview submission PDF should be uploaded separately; this source package is provided for reproducibility and last-mile edits.
